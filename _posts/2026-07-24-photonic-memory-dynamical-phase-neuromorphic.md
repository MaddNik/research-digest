The paper proposes that three distinct forms of photonic memory, dispersive, nonlinear, and driven-dissipative, can all be understood as manifestations of a single underlying mechanism: evolution of optical phase over time. Dispersive memory is attributed to frequency-dependent phase accumulation, nonlinear memory to intensity-dependent phase evolution, and driven-dissipative memory to convergence toward an attractor state. The authors validate this framework experimentally using silicon photonic waveguides patterned with chirped Bragg gratings, measuring distinct linear and nonlinear memory responses in a single integrated device. They describe this as the first experimental demonstration showing all three photonic memory regimes coexisting on one platform, with implications for neuromorphic photonics and reservoir computing.
